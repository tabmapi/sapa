# SAPA
This repository contains the files necessary for running symmetry-adapted PDF analysis with Topas v6 and v7.

## Requirements
* Topas v6 or v7
* Python 3.6 or later
* HDF5
* Topas_for_PDF macros - available at github.com/pachater/topas
